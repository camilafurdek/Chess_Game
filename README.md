# Chess_Game

A chess game built with Python using Pygame featuring full move validation, special rules, and a graphical board with drag-and-drop pieces.

---

## Features

- Complete chess board with graphical interface
- Drag and drop piece movement
- Valid move calculation and check detection
- Castling, en passant, and pawn promotion
- Move and capturing sound effects
- Diferent color themes
- Last move history

---

## Technologies used

- Python 3
- Pygame

## How to run

1. Clon repository
   ```bash
      $ git clone https://github.com/camila.furdek/chess_game.git
   ```
2. Go into the folder
   ```bash
      $ cd chess_game
   ```
3. Install dependencies
    ```bash
      $ pip install pygame
   ```
4. Run the game
   ```bash
      $ python src/main.py
   ```

---

## Controls

- Click on a pice to see al valid moves, drag it to move it
- Press r to re-start the game
- Press t to change color theme
